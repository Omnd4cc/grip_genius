# Grip Genius

一个简洁好用的攀岩助手 H5 应用。提供 Beta 分析和攀岩日结记录功能。

## 功能介绍

- **Beta 分析**: 记录并分析你的攀岩 Beta，优化动作细节。
- **攀岩日结**: 记录每日攀岩训练内容，生成日结报告。

## 开发指南

### 环境准备

确保你的本地开发环境已安装 [Node.js](https://nodejs.org/) (推荐 v16+)。

### 安装依赖

克隆项目后，在项目根目录下运行以下命令安装依赖：

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

启动后，访问终端中显示的本地地址（通常是 `http://localhost:5173`）即可预览应用。

### 构建生产版本

```bash
npm run build
```

构建产物将输出到 `dist` 目录。

### 预览生产版本

```bash
npm run preview
```
